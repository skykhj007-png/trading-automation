# V35 Signal Pro 수동 매매 가이드

## 핵심 원칙

**자동매매 승률**: 17~35% (실패)
**수동 판단**: 시그널 + 본인 확인 = 높은 승률 기대

---

## 시그널 해석

### 강력 진입 (MTF 시그널)

| 조건 | 설명 |
|------|------|
| 큰 라임 ▲ | 현재TF + HTF 동시 LONG |
| 큰 핑크 ▼ | 현재TF + HTF 동시 SHORT |

**MTF 시그널 = 가장 신뢰도 높음**

### 일반 진입

| 조건 | 설명 |
|------|------|
| 초록 ▲ | 현재TF LONG (HTF 확인 필요) |
| 빨강 ▼ | 현재TF SHORT (HTF 확인 필요) |

---

## 진입 전 체크리스트

### LONG 진입 조건

```
□ 추세: UP 또는 up
□ HTF: 상승 또는 LONG
□ ADX: 20 이상 (추세 존재)
□ RSI: 30~50 (과매도 탈출 구간)
□ StochK: 20~40 (과매도 탈출)
□ MACD: BULL
□ 거래량: 1.5x 이상 (!)
□ 매수압력: 55% 이상
□ BB: 하단 터치 또는 중앙↑
```

**7개 이상 충족 시 진입**

### SHORT 진입 조건 (비추천)

```
□ 추세: DOWN 또는 down
□ HTF: 하락 또는 SHORT
□ ADX: 20 이상
□ RSI: 50~70 (과매수 진입 구간)
□ StochK: 60~80 (과매수)
□ MACD: BEAR
□ 거래량: 1.5x 이상
□ 매수압력: 45% 이하
□ BB: 상단 터치 또는 중앙↓
```

---

## TP/SL 가이드

### 1분봉
- TP: 0.3~0.5%
- SL: 0.2~0.3%
- R:R = 1.5:1

### 15분봉
- TP: 0.5~1.0%
- SL: 0.3~0.5%
- R:R = 2:1

### 1시간봉
- TP: 1.0~2.0%
- SL: 0.5~1.0%
- R:R = 2:1

---

## 진입 타이밍

### 최적 진입
1. MTF 시그널 발생
2. RSI/StochK 과매도에서 상승 시작
3. 거래량 급증 (1.5x+)
4. 양봉 캔들 확인

### 피해야 할 상황
1. HTF와 현재TF 방향 불일치
2. 거래량 낮음 (1x 이하)
3. ADX 15 이하 (추세 없음)
4. RSI 50 근처 (애매한 구간)

---

## 실전 예시

### LONG 진입 예시
```
시그널: MTF LONG 18점
━━━━━━━━━━━━━━━━
추세: UP        ✓
HTF 60: LONG    ✓
ADX: 25!        ✓
RSI: 35         ✓ (과매도 탈출)
StochK: 28      ✓ (상승 중)
MACD: BULL      ✓
거래량: 2.1x!!  ✓
매수압력: 62%   ✓
BB: 하단        ✓

→ 9/9 충족 = 강력 진입
→ 진입가: $94,500
→ TP: $94,785 (+0.3%)
→ SL: $94,310 (-0.2%)
```

### 진입 보류 예시
```
시그널: LONG 15점
━━━━━━━━━━━━━━━━
추세: up        ✓
HTF 60: 횡보    ✗ (불일치)
ADX: 18         ✗ (약함)
RSI: 48         △ (애매)
StochK: 52      ✗ (중립)
거래량: 0.9x    ✗ (낮음)

→ 3/9 충족 = 진입 보류
→ 다음 시그널 대기
```

## 핵심 요약

1. **MTF 시그널만 진입** (현재TF + HTF 일치)
2. **체크리스트 7개 이상 충족** 시 진입
3. **거래량 확인 필수** (1.5x 이상)
4. **SHORT는 피하고 LONG 위주**
5. **손절은 빠르게, 익절은 느긋하게**
