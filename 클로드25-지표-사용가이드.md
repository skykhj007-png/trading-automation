## 1. 지표 개요

### 지원 모드
| 모드 | 신호 | 특징 |
|------|------|------|
| **선물** | LONG + SHORT | 양방향 매매, 레버리지 연동 TP/SL |
| **현물코인** | LONG만 | 매수만, EXIT 경고 제공 |
| **주식** | LONG만 | 매수만, 스윙 트레이딩용 |

### 사용 지표
- 이동평균선 (MA 50, 200, EMA 9/21/55)
- VWAP 100
- Stochastic RSI
- Bollinger Bands
- ADX (추세 강도)
- 거래량 분석 (고래/기관 감지)
- 멀티타임프레임 (MTF) 분석

## 5. 신호 조건

### LONG 신호 조건
1. 불장단타왕 점수 >= 5점
2. 기본 신호 조건 충족 (EMA 크로스 + RSI)
3. 총점 >= 최소 신호 강도 (기본 10점)
4. StochRSI 과매도 또는 골든크로스
5. (옵션) MTF 상승 추세 확인
6. (옵션) ADX 추세 존재 + DI+ 우위

### SHORT 신호 조건 (선물만)
1. 기본 신호 조건 충족 (EMA 크로스 + RSI)
2. 상승추세 아님 (MA 역배열)
3. 가격 < MA200
4. 총점 >= 최소 신호 강도
5. StochRSI 과매수 또는 데드크로스
6. (옵션) MTF 하락 추세 확인

### 신호 필터
- **ADX 필터**: 횡보장 (ADX < 18) 신호 차단
- **OI 필터**: 포지션 증가 시에만 진입
- **펀딩비 필터**: 극단적 과열 시 반대 포지션 주의

## 7. 실전 사용법

### 진입 체크리스트

#### LONG 진입 전 확인
- [ ] 총점 10점 이상인가?
- [ ] EMA 정배열인가? (9 > 21 > 55)
- [ ] 고래가 매수 중인가?
- [ ] 상위 시간대도 상승 추세인가?
- [ ] StochRSI가 과매도 구간인가?

#### SHORT 진입 전 확인 (선물)
- [ ] 총점 10점 이상인가?
- [ ] EMA 역배열인가? (9 < 21 < 55)
- [ ] 고래가 매도 중인가?
- [ ] 가격이 MA200 아래인가?
- [ ] StochRSI가 과매수 구간인가?

### 권장 진입 시점
1. **9/21 골든크로스** 발생 + 점수 10점 이상
2. **BB 하단 터치** + 과매도 + 고래 매수
3. **불리시 다이버전스** + 지지선 근처
4. **50/200 골든크로스** 발생 (장기 강세 전환)

### 권장 청산 시점
1. **TP1 도달** → 50% 익절
2. **TP2 도달** → 나머지 익절
3. **9/21 데드크로스** 발생 → 청산 고려
4. **고래 매도 감지** → 청산 고려
5. **SL 도달** → 손절

### 피해야 할 상황
- ADX < 18 (횡보장)
- 고래와 추세 방향 불일치
- 점수 10점 미만
- BB 중앙에서 진입
- 상위/하위 시간대 추세 불일치

## 8. 자주 묻는 질문

### Q: 신호가 너무 자주/드물게 나와요
**A:** 설정에서 "최소 신호 강도"를 조정하세요.
- 더 많은 신호: 8-10점
- 더 정확한 신호: 12-15점

### Q: EMA 크로스 표시가 너무 많아요
**A:** 코드에서 `plotshape(ema9_21_goldenCross...)` 부분을 주석처리하면 숨길 수 있습니다.

### Q: 어떤 시간대에서 사용해야 하나요?
**A:**
- 스캘핑: 1분, 3분, 5분
- 단타: 15분, 1시간
- 스윙: 4시간, 일봉

### Q: 고래 감지가 정확한가요?
**A:** 거래량이 평균의 2배(고래) 또는 3배(기관) 이상일 때 감지합니다. 실제 고래 지갑 추적은 아니지만 큰 거래량은 대형 참여자를 의미합니다.

### Q: 레버리지 설정은 어떻게 하나요?
**A:**
- 초보자: 5-10x
- 중급: 10-25x
- 고급: 25-50x
- ⚠️ 50x 이상은 매우 위험

### Q: MTF 분석은 어떤 시간대를 보나요?
**A:**
| 모드 | HTF | MTF | LTF |
|------|-----|-----|-----|
| 선물 | 1시간 | 15분 | 5분 |
| 현물/주식 | 일봉 | 4시간 | 1시간 |
