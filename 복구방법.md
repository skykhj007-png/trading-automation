# 실수했을 때 복구 방법

## 1. 파일 하나만 복구
```bash
git checkout HEAD -- 파일이름
```

예시:
```bash
git checkout HEAD -- cloudflare-worker.js
git checkout HEAD -- 매매지표-V39-MTFConfluence.pine
```

## 2. 모든 변경 취소 (마지막 커밋으로)
```bash
git checkout HEAD -- .
```

## 3. 이전 버전 보기
```bash
git log --oneline
```

## 4. 특정 커밋으로 복구
```bash
git checkout [커밋ID] -- 파일이름
```

## 5. 백업 실행
더블클릭: `auto-backup.bat`
