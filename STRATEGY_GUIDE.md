# 🎯 선물 트레이딩 통합 전략 가이드

## 📊 전략 개요

이 시스템은 **불장단타왕 전략**과 **클로드21 멀티타임프레임 전략**을 결합하여 선물 거래의 승률을 극대화합니다.

## 🔥 핵심 강점

### 1. 이중 검증 시스템
- 두 전략이 **모두 동의**할 때만 진입
- 잘못된 신호(False Signal) 최소화
- 승률 우선, 거래 빈도는 낮을 수 있음

### 2. 점수 기반 시스템
- **최대 24점** 만점 (불장단타왕 10점 + 클로드21 8점 + 필터 6점)
- 기본 임계값: **12점** (조정 가능)
- 점수가 높을수록 신호 신뢰도 높음

### 3. 분할 익절
- TP1: +0.8% (50% 물량 익절)
- TP2: +1.5% (나머지 50% 익절)
- SL: -0.3% (철저한 손절)

---

## 📈 전략 1: 불장단타왕 (10점)

### 핵심 지표
1. **VWAP 100선** (3점)
   - 거래량 가중 이동 평균 100
   - 가격이 VWAP 100에 닿을 때 ±1% 이내 = 3점

2. **터닝 포인트** (5점)
   - 하락 추세에서 반등 포착
   - 이평선이 완만해지고 캔들이 뚫고 올라감
   - 조건:
     * VWAP/MA 기울기 완만 (3점)
     * 최근 5개 중 3개 이상 상승 (3점)
     * VWAP 골든 크로스 (5점)
     * 정배열 (MA7 > MA15 > MA50) (4점)
   - 8점 이상 = 터닝 포인트 인정

3. **정배열 확인** (2점)
   - MA7 > MA15 > MA50

### 손익 기준 (스윙)
- 익절: +7%
- 손절: -1%
- **손익비 1:7**

---

## 🎯 전략 2: 클로드21 멀티타임프레임 (8점)

### 멀티 타임프레임 분석

#### 15분봉 - 주요 트렌드 (3점)
- EMA 20 > EMA 50
- RSI: 45~70 (과매수 회피)
- 조건 만족 시 3점

#### 5분봉 - 중간 트렌드 (2점)
- EMA 20 > EMA 50
- MACD 상승 중
- RSI > 40
- 조건 만족 시 2점

#### 3분봉 - 진입 신호 (3점)
- EMA 20/50 골든 크로스
- MACD Line > Signal Line
- 조건 만족 시 3점

#### 1분봉 - 보조 지표
- RSI: 35~65 (극단 회피)
- 가격 > VWMA 20

### 신호 강도
- **기본 신호**: 3분봉 크로스오버 = 3점
- **15분 트렌드 일치**: +3점
- **5분 트렌드 일치**: +2점
- **최대**: 8점

---

## 🔍 전략 3: 추가 필터 (6점)

승률 향상을 위한 추가 검증

### 1. 평균 회귀 확인 (2점)
- 현재가와 VWAP 100 이격도 체크
- ±10% 이내 = 정상 (2점)
- ±10% 초과 = 과매수/과매도 (0점, 진입 회피)

### 2. POC 지지 확인 (2점)
- Volume Profile Point of Control
- 가장 많은 거래량이 발생한 가격대
- 현재가가 POC ±2% 이내 = 2점

### 3. 200선 위치 (2점)
- 현재가 > MA 200 = 상승 추세 확인 (2점)

## 💰 진입 조건

### LONG 진입
```
✅ 필수 조건:
1. 불장단타왕 점수 >= 5점
2. 클로드21 신호 = LONG
3. 종합 점수 >= 12점 (MIN_SIGNAL_STRENGTH)

✅ 이상적인 진입 (15점 이상):
- VWAP 100 터치 (3점)
- 터닝 포인트 (5점)
- 정배열 (2점)
- 15분/5분 모두 상승 트렌드 (5점)
- 3분 EMA 크로스 (3점)
- 추가 필터 통과 (2~6점)
```

### SHORT 진입 (보수적)
```
⚠️ 조건:
1. 클로드21 신호 = SHORT
2. 불장단타왕 역배열 (MA7 < MA15 < MA50)
3. 현재가 < MA 200
4. 종합 점수 >= 12점

⚠️ 주의: SHORT 점수는 0.8배 적용 (더 보수적)
```

## 📉 손익 관리

### 분할 익절 전략

| 단계 | 목표 | 청산 비율 | 설명 |
|------|------|-----------|------|
| TP1 | +0.8% | 50% | 빠른 수익 실현 |
| TP2 | +1.5% | 50% | 트렌드 추종 |
| SL | -0.3% | 100% | 빠른 손절 |

### 추가 손절 조건
1. **VWAP 100 하향 돌파**
   - 이전 캔들 >= VWAP 100
   - 현재 캔들 < VWAP 100
   - 즉시 전량 손절

2. **-0.3% 도달**
   - 목표 손실률 도달 시 즉시 손절

## ⚙️ 설정 가이드

### 공격적 설정 (높은 거래 빈도)
```javascript
MIN_SIGNAL_STRENGTH: 10  // 임계값 낮춤
STRICT_MODE: false       // 완화 모드
TP1_PERCENT: 0.5         // 빠른 익절
TP2_PERCENT: 1.0
STOP_LOSS_PERCENT: 0.5   // 여유 있는 손절
```

### 기본 설정 (균형)
```javascript
MIN_SIGNAL_STRENGTH: 12  // 기본값
STRICT_MODE: true        // 엄격 모드
TP1_PERCENT: 0.8
TP2_PERCENT: 1.5
STOP_LOSS_PERCENT: 0.3
```

### 보수적 설정 (높은 승률)
```javascript
MIN_SIGNAL_STRENGTH: 15  // 매우 높은 임계값
STRICT_MODE: true        // 엄격 모드
TP1_PERCENT: 1.0
TP2_PERCENT: 2.0
STOP_LOSS_PERCENT: 0.2   // 빠른 손절
```

## 🎓 사용 팁

### 1. 타임프레임 우선순위
- **15분봉** > **5분봉** > **3분봉**
- 큰 프레임이 더 중요함
- 15분봉 트렌드와 반대 방향은 진입하지 마세요

### 2. VWAP 100의 역할
- **지지선**: VWAP 위에서 매수
- **저항선**: VWAP 아래에서 매도
- **손절선**: VWAP 깨지면 즉시 탈출

### 3. 터닝 포인트 포착
- 하락 후 처음 반등하는 순간이 가장 좋음
- 이평선이 "완만"해지는 것 관찰
- 너무 늦으면 추격 금지

### 4. 과매수/과매도 회피
- RSI 극단값 회피
- VWAP에서 ±10% 이상 이격 시 진입 금지
- 평균 회귀를 기다림

## ⚠️ 주의사항

1. **과도한 레버리지 금지**
   - 권장: 5배 이하
   - 초보자: 2~3배

2. **신호 강도 무시 금지**
   - 12점 미만은 절대 진입하지 마세요
   - "그냥 들어가보자"는 도박입니다

3. **손절 지키기**
   - VWAP 깨지면 무조건 손절
   - "조금만 더 기다려보자"는 금물

4. **감정 거래 금지**
   - 손실 후 복수 거래 금지
   - 연패 시 하루 쉬기
   - 연승 시 레버리지 올리지 않기
