## 신호 발생 조건

### LONG 신호
1. 점수 >= 최소 신호 강도
2. 엄격 모드 시: 불장단타왕 >= 6점 AND 클로드21 >= 4점
3. 멀티TF 필터 시: 상위 타임프레임 확인
4. ADX 필터 시: 추세 존재 확인

### SHORT 신호 (선물만)
1. 점수 >= 최소 신호 강도 (반대 조건)
2. 엄격 모드 시: 조건 충족
3. 거래 모드 = "선물"

### EXIT 신호
- 고래 매도 감지 시 경고

## 문제 해결

### 신호가 안 나오는 경우
1. 최소 신호 강도 낮추기 (10~12)
2. 엄격 모드 OFF
3. ADX 필터 OFF

### 신호가 너무 많은 경우
1. 최소 신호 강도 높이기 (14~16)
2. 엄격 모드 ON
3. ADX 필터 ON

### 승률이 낮은 경우
1. 최소 신호 강도 높이기
2. 멀티TF 필터 ON
3. 고래 감지 확인 후 진입
